# 🚀 Git-Score – GitHub Analyzer

Git-Score is a full-stack web application that analyzes GitHub profiles and evaluates developers using a smart scoring system (0–100). It provides insights, role detection, and leaderboards to compare developers globally.

---

## 🌐 Live Demo

- Project URl : https://git-score-sandy.vercel.app/  
  

---

## ✨ Features

- 🔍 Analyze any GitHub profile
- 📊 Developer score (0–100) based on:
  - Repositories
  - Stars
  - Activity
  - Consistency
- 🤖 AI-powered insights using Google Gemini
- 🧠 Role detection (Full Stack, Backend, etc.)
- 🏆 Global & personal leaderboard
- ❤️ Like / 🔖 Save profiles
- 📈 Activity & language analysis
- 🔐 Authentication (JWT + cookies)
- 📂 Personalized dashboard
